# Privacy Policy — Aurora OCR/TTS Extension

**Last updated:** 1/16/2026

This privacy policy describes how the Aurora OCR/TTS Chrome extension (“Extension”) handles data.

## Summary
The Extension only sends user‑selected content to a user‑configured Aurora OCR/TTS server. It does not sell or share data with third parties.

## Data We Collect
- **Website content (user‑selected text or images):** Sent to the Aurora server you configure in the Extension to perform OCR/TTS.

We do **not** collect:
- Personally identifiable information
- Health or financial data
- Authentication credentials
- Location data
- Web history beyond the selected content

## How We Use Data
- To provide OCR and text‑to‑speech processing via your configured Aurora server.

## Data Sharing
- Data is sent only to the Aurora server you configure. We do not share data with third parties.

## Data Storage
- The Extension stores user preferences (such as server URL and settings) locally in the browser.

## Security
- Data is transmitted only to your configured server. You control where the server is hosted.
